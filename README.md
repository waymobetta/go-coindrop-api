# go-coindrop-api

> The CoinDrop API

## Getting started

Start server

```bash
make start
```

### Environments

add env vars to `.env.local`, then run:

```bash
make start/local
```

add env vars to `.env.staging`, then run:

```bash
make start/staging
```

add env vars to `.env.prod`, then run:

```bash
make start/prod
```
